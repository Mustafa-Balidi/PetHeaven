# 🐾 Pet Haven

**Pet Haven** is a comprehensive graduation project designed to bridge the gap in pet care services. It unifies four main pillars into one digital ecosystem:

- 🏡 **Adoption Hub:** A transparent and automated pet adoption system with user compatibility quizzes.
- 🛒 **Smart Store:** An integrated e-commerce platform for categorized pet supplies.
- 🤖 **AI Symptom Checker:** An NLP-based tool (Python/Flask) providing immediate preliminary health guidance for pets.
- 🩺 **Vet Booking:** A GPS-integrated system to find and book nearby veterinarians.

### 🛠️ Tech Stack:
`React.js` | `Tailwind CSS` | `ASP.NET Core` | `SQL Server` | `Python / Flask`
